# Прототип — Универсальный лендинг (v1)

> Работает под любой креатив. Классическая блочная структура, но с живыми текстами.

---

## 1 | Hero
**Модуль ВК:** изображение
**Визуал:** фото с тренировки (энергия, движение, реальные девушки). Логотип InStretch.

```
Попробуй 2 тренировки за 490 ₽

Женская студия фитнеса и растяжки в Брянске.
12 направлений. Группы до 8 человек.

↓ Листай
```

---

## 2 | Оффер
**Модуль ВК:** изображение
**Визуал:** контрастная карточка оффера.

```
2 первые тренировки — 490 ₽
                      (было ~1100 ₽)

Без абонемента. Без обязательств.
Просто приходи и попробуй.
```

## 3 | Боли — «Знакомо?»
**Модуль ВК:** изображение
**Визуал:** мягкий фон, крупный текст.

```
Знакомо?

✓ Сидишь целый день — спина ноет
✓ Хочешь быть гибкой, но «я деревянная»
✓ Начинаешь и бросаешь — нет мотивации
✓ В общем зале некомфортно
✓ Хочешь подтянуть тело, но тренажёрка — не твоё

Всё это решается.
```

## 4 | Решение — что такое InStretch
**Модуль ВК:** изображение
**Визуал:** фото интерьера (бирюзовые гамаки, чистый зал) + иконки.

```
InStretch — студия только для девушек

Ни одного мужика. Расслабься.

◆ Мини-группы до 8 человек —
  тренер видит тебя, а не спину 30-го в зале
◆ 12 направлений в одном абонементе —
  выбирай тренировку по настроению
◆ 2 студии — Бежица и Советский район
◆ 5.0 на Яндекс Картах — 216 отзывов
```

## 5 | Направления
**Модуль ВК:** карусель (каждое направление = слайд)
**Визуал:** фото с тренировки + название + 1 строка пользы.

**Слайд 1:**
```
Bungee фитнес

Кардио на тросах — сжигаешь калории
и даже не замечаешь.
Выносливость, сила, кайф.

```
Стретчинг

Гибкость, лёгкость, шпагат.
Подходит всем — даже если
никогда не тянулась.
```

**Слайд 3:**
```
Lady Dance

## 6 | Видео
**Модуль ВК:** видео, 15–30 сек

Нарезка с разных тренировок. Акцент: эмоции, улыбки, обычные девушки (не фитоняшки). Финал — логотип + «Приходи попробовать».
